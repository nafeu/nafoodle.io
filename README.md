# nafoodle.io

Experimental multiplayer party game starter built with React, NodeJs, and Socket.io.

## Development

- This project is an early work-in-progress...

```
git clone https://github.com/nafeu/nafoodle.io.git
cd nafoodle.io
nvm use v12.14.1
npm install
```

### Run Server

```
npm run dev
```

### Client

```
cd client
npm start
```
